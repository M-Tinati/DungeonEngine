# 🚀 فلسفه پروژه

هدف **Dungeon Engine** ساخت یک بازی تجاری نیست.

این پروژه به‌عنوان یک **مسیر یادگیری پروژه‌محور (Project-Based Learning)** طراحی شده است تا هر مفهوم جدید، در قالب حل یک مسئله واقعی پیاده‌سازی شود و تجربه‌ای نزدیک به توسعه نرم‌افزار در دنیای واقعی ایجاد کند.
